What this is

This page is served by nginx, running in a Docker container, on a pod, on a Kubernetes cluster (AWS EKS). The infrastructure was provisioned with Terraform. Every time I push a change to this page, a GitHub Actions pipeline builds a new image, pushes it to ECR, and rolls it out to the cluster automatically.

Live pipeline status

What's running

  • Terraform - provisions the VPC, EKS cluster, and node group. State stored in S3, locked with DynamoDB.
  • GitHub Actions - authenticates to AWS via OIDC (no stored access keys). Runs plan/apply for infra and build/deploy for this app.
  • Prometheus + Grafana - installed via Helm, watches pod and node health across the cluster.
  • This page - a static site in an nginx container, rebuilt and redeployed on every commit.
